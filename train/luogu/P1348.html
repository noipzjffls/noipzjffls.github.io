P1348 Couple number

我们可以很容易地注意到，该题目要求我们在[n1,n2]区间找到满足题意的Couple number N。

由题意可知，N=a^2-b^2(N∈[n1,n2]且a,b∈Z)。

而a^2-b^2=(a+b)(a-b)。

假设a-b=k，则a=b+k。即k(k+2b)=N。我们可以注意到，k和k+2b奇偶性必定相同。

当k∈{x|x=2k+1,k∈Z}时，k+2b∈{x|x=2k+1,k∈Z}。所以N∈{x|x=2k+1,k∈Z}。在这个条件下，只要[n1,n2]区间的一个数m满足m∈{x|x=2k+1,k∈Z}，则m为Couple number。

当k∈{x|x=2k,k∈Z}时，k+2b∈{x|x=2k,k∈Z}。所以N∈{x|x=4k,k∈Z}。在这个条件下，只要[n1,n2]区间的一个数m满足m∈{x|x=4k,k∈Z}，则m为Couple number。

∴综上所述，能被4整除的数和一切奇数都是Couple number。

#include<cstdio>

long long n1,n2,ans;//定义长整型数

int main()

{

    scanf("%lld%lld",&n1,&n2);

    for(long long i=n1;i<=n2;i++)//在[n1,n2]区间查找

    {

        if(i&1||(((i&1)==0)&&((i/2)&1)==0))

        //说明：&为按位与，是一种位运算。其计算原理是将两个被操作数转换为二进制，然后看数码都为1的个数。这里是i与1进行按位与，而1转换为二进制为1，因此结果最多为1。这个时候就要看i在二进制下的末位是否为1，而十进制偶数转换为二进制后末位必定为0，只有i为奇数时，i&1才能为真，因此i&1作为条件时等价于i&1==1和i%2==1。

        {

            ans++;//答案加1

        }

    }

    printf("%lld",ans);

}

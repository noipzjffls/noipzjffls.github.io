题目的本质就是让我们输出两个数再输出两个数的和。因此我们可以得到代码：

#include<iostream>//头文件

using namespace std;//引用名字空间

int a,b;//定义两个变量

int main()//主函数

{

    cin>>a>>b;

    cout<<a+b;

    return 0;

}

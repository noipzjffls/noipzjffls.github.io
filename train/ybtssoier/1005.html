第1005题：地球人口承载力估计

该题是一道经典的牛吃草（牛顿）问题，这种题目在数学题中出现也较多，但在编程中也会“崭露头角”。

我们输入了四个整数，而用这四个整数，我们可以算出(b-a)年里资源增长量，即b*y-a*x。

接下来我们要算出每年平均资源增长量，即(b*y-a*x)/(b-a)。

题目要求我们求出地球最多可养活几亿人，“最多”就代表资源增长量为0，因此此时可以养活(b*y-a*x)/(b-a)。

最后输出的是浮点数，因此最终结果的表达式为1.0*(b*y-a*x)/(b-a)，输出格式为double或者float。

#include<cstdio>

using namespace std;

int x,a,y,b;

double z;

int main()

{

    scanf("%d%d%d%d",&x,&a,&y,&b);

    z=1.0*(b*y-a*x)/(1.0*(b-a));

    printf("%.2lf",z);

    return 0;

}

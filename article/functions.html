关于C++函数初步

ericoder

我们知道，C++源代码是与函数分不开的。哪怕是最简单的Hello World的无输入程序也需要main函数的参加。

那么，函数有哪些用途呢？

1、整个程序的包装

像上文所说的，任何一个C++程序都需要main函数，main函数起一个包装的作用。

2、使程序表述更加清晰

我们有时可能会多次运用到同一个（组）语句，我们这时可以将它放入函数（一般为void形式，指不返回任何值）

3、进行main函数无法完成的任务

递归、搜索是函数价值的体现。这些功能在main函数的单独参与下是难以完成的。

对于1，我们一般会有：

#include<iostream>//头文件

using namespace std;//引用名字空间

int main()//主函数

{

    cout<<"Hello world!"<<endl;

    return 0;

}

这是一个最基本的一个程序，我们来对其进行分析：

（1）虽然数据结构（变量等）不需要头文件，但如果没有头文件，那么就无法输入输出，这样程序的意义何在？（就像在工厂中可以直接完成加工，但是没有原材料，也没有最终产品上市）

（2）名字空间一般为std，std是standard（标准）的缩写。cin、cout、STL功能等均需要使用。如果不加using namespace std;，那么就需要在cout、endl前面加上std::，如：

#include<iostream>//头文件

int main()//主函数

{

    std::cout<<"Hello world!"<<std::endl;

    return 0;

}

（3）主函数一般前面加int，为返回值；程序最后要加return 0;语句。



对于2，我们可以有：

#include<iostream>//头文件

using namespace std;//引用名字空间

int a,b;//定义变量

void print(int x)

{

    cout<<x<<endl;

}

int main()//主函数

{

    cin>>a>>b;

    print(a);//调用函数，并将变量代入

    print(b);//同上

    return 0;

}

输入：3 24

输出：

3

24

Program ended with exit code: 0

对于3，我们可以使用递归，如：

#include<iostream>//头文件

using namespace std;//引用名字空间

int a;//定义变量

int f(int x)

{

    if(x==0)return 1;

    return f(x-1)*x;

}

int main()//主函数

{

    cin>>a;

    cout<<f(a);//输出a的阶乘

    return 0;

}

这个程序是输出a的阶乘的，是通过函数自己调用自己实现的（第7行）。注意第4行有int（代表函数返回类型），说明函数中的return后面要跟int类型的数据，而如果是void，则需写return;。

输入：5

输出：

120

Program ended with exit code: 0
